# 受付

今回用のため、次回があるときは次回の意思（香典辞退など）による。

## 流れ

1. 弔問客（ちょうもんきゃく）
   > 参列のお礼を述べる。（明るすぎず、落ち着いた丁寧なトーン）
   > 「本日はお忙しいところお越しいただき、ありがとうございます。」※基本
   > 「お足元の悪い中お越しいただき、ありがとうございます。」※天気の悪いとき
2. 芳名カードのご記入
   > 「まずはご記帳所にて、ご芳名をいただいてもよろしいでしょうか」※芳名カードのとき
   > 「こちらにご記帳いただけますでしょうか」※芳名帳のとき
   > 名刺の場合は、催促不要
3. 香典の受け取り
   > 「お預かりいたします」と一言を添えて**両手**で受け取る
   > 受け取った後に一礼する。
4. 返礼品を渡す
   > 「こちらをお持ちください」「こちらをどうぞ」と言葉を添えて**両手で**渡す
   > 終了時に渡す場合
   > 「お帰りの際に返礼品をお渡しいたします。こちらの引換券はお帰りの際まで大切にお持ちください。」
5. 弔問客が離れたら会計係へ渡す

6. 会場への案内
   1. 会場の入り口をご案内
      1. 開始期まで時間があり、お線香などが用意されている場合
         > 「お線香をお手向けいただけますので、会場までお進みください」
   2. ご高齢のかたにはサポート
   3. クロークがある場合
      1. コートやお手荷物などを預かり、引換券をお渡しする。貴重品はあずかれない旨の説明。



## 弔意（ちょうい）への返し

| 参列者の言葉          | 受付での返し（定型） |
| --------------        | ---------- |
| ご愁傷さまです        | 恐れ入ります |
| ご愁傷さまです        | お心遣いありがとうございます。 |

❌「大丈夫です」  
❌「慣れています」  
❌「気にしないでください」  
❌ 長い説明・感情的な返し
